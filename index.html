Latefy

Add what is playing to Deezer

View the Code on GitHub

Latefy is an iOS app that analyzes the currently playing music, finds out which track is playing and adds it to your Deezer account.

Latefy started as a hack at the Deezer BeMyApp Hackathon in Berlin in July 2012. It was made by Johan (@freenerd), Michael and Stefan (@hintz). For fingerprinting the Gracenote API is used.

We will probably bring it to the iOS App Store soon.
